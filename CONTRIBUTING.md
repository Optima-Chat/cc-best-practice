# 贡献指南

感谢你对 Claude Code 最佳实践项目的关注！

## 如何贡献

### 提交最佳实践

如果你有使用 Claude Code 的最佳实践想要分享：

1. Fork 本仓库
2. 编辑 `README.md`，添加你的最佳实践
3. 提交 Pull Request

### 内容要求

- 基于真实使用经验
- 表述清晰简洁
- 适合开发专家阅读

### PR 规范

- PR 标题简洁明确
- 说明添加的内容和原因
- 一个 PR 解决一个问题

### 报告问题

发现文档错误或有改进建议？欢迎[创建 Issue](https://github.com/Optima-Chat/cc-best-practice/issues)。

## 许可证

贡献内容将使用 MIT 许可证。
